一、 实现元素居中

1.1 calc 实现

body {
  padding: 0 calc(50% - 500px);
}
#app {
  width: 1000px;
  height: 1000px;
  background-color: cyan;
}

1.2. margin + absolute 实现

body {
   width: 100%;
   height: 100vh;
 }

 #app {
   width: 500px;
   height: 500px;
   position: absolute;
   top: 0;
   left: 0;
   bottom: 0;
   right: 0;
   margin: auto;
   background-color: cyan;
 }

1.3 absolute 实现

body {
  width: 100%;
  height: 100vh;
}
#app {
  width: 500px;
  height: 500px;
  position: absolute;
  top: 50%;
  left: 50%;
  transform: translate(-50%, -50%);
  background-color: cyan;
}

1.4 flex 实现

body {
   width: 100%;
   height: 100vh;
   display: flex;
   justify-content: center;
   align-items: center;
 }
 #app {
   width: 500px;
   height: 500px;
   background-color: cyan;
 }

二、 1px问题

-产生原因：

  • 设备像素比 DPR,当设计稿是1px时,是物理像素,而我们写的1px是css像素,而根据DPR,展示到页面上就是2px,比原来粗了,其实应该设置0.5px css像素
  • 解决
    • 新项目
      // 在html中,获取当前设备的DPR,根据DPR,修改viewport的属性
// 获取 meta标签
var viewport = document.querySelector("meta[name=viewport]")
// 拿到当前设备的DPR
var dpr = window.devicePixelRatio
// 修改他的属性
viewport.setAttribute('content', `width=device-width, initial-scale=${1 / dpr}, maximum-scale=${1 / dpr}, minimum-scale=${1 / dpr}, user-scalable=no`)
// 拿到HTML
var docEl = document.documentElement;
// 设置 HTML的 fontsize 750/320
var fontsize = 10 * (docEl.clientWidth / 320) + 'px';
docEl.style.fontSize = fontsize;
    • 伪类 + transform

三、css3 新特性

3.1 边框属性

  • border-radius 设置圆角
  • border-shadow 设置边框阴影 阴影的X–阴影的Y—阴影的模糊值—阴影的颜色—inset内/默认外
  • border-image 设置边框图片,必须先写border,再加图片
    #app {
  width: 100px;
  height: 100px;
  border: 20px red solid;
  border-image: url(../images/girl-01.jpg) 30 30 repeat;
}
  • border-radius 设置边框圆角

3.2 背景

  • background-image 设置背景图片
  • background-size 设置背景图像的大小
  • background-origin 设置背景图像的位置区域
  • background-clip 背景从哪开始设置
    • content-box 只包含内容
    • padding-box 包含了内容和padding

3.3 渐变

  • 线性渐变

    • linear-gradient 默认是 从上到下 to bottom

  • 径向渐变

    • radial-gradient
      background: radial-gradient(circle, red, blue, yellow)

3.4 文本属性

  • text-shadow 设置文本的阴影
  • box-shaddow 设置盒子阴影
  • text-overflow 设置文本溢出属性
  • word-wrap 设置文本是否换行
  • word-break 设置非中文的换行规则,可以让英文换行

3.5 2D转换

  • transform
    • translate 平移x、y
    • rotate 旋转 deg
    • scale 缩放(X,Y) 默认宽、高 1
    • skew X、Y倾斜的角度
    • matrix 合并上面的方法

3.6 3D转换

  • 给父盒子设置 

     perspective: 150;
-webkit-perspective: 150;

  • 利用transform的3d进行处理

3.7 过渡 transition

  • transition 需要触发设置的属性改变,才能执行
    • 设置修改的css属性
    • 执行多长时间
    • 速度
    • 延时执行 默认为0
      #app {
  width: 500px;
  height: 500px;
  border: 20px green solid;
  position: relative;
  border-image: url(../images/girl-01.jpg) 30 30 repeat;
  padding: 20px;
  transition: width 2s;
  -webkit-transition: width 2s;
}
#app:active {
  width: 1000px;
}

3.8 animation 动画

  • animation
    • name 动画的名字
    • duration 执行时间
    • timing-function 速度
    • delay 延时多长时间执行
    • iteration-count 播放次数
    • direction 方向 默认正常,alternate反向
    • fill-mode 动画不执行的样式
      animation: name duration timing-function delay iteration-count direction fill-mode;
      #app {
  width: 500px;
  height: 500px;
  border: 20px green solid;
  position: relative;
  border-image: url(../images/girl-01.jpg) 30 30 repeat;
  padding: 20px;
  background: brown;
  animation: myanimation 5s linear 2s  infinite;
}
@keyframes myanimation {
  0% {
    background: darkmagenta;
  }

  25% {
    background: bisque;
  }

  50% {
    background: yellowgreen;
  }

  75% {
    background: palevioletred;
  }

  100% {
    background: blue;
  }
}

3.9 box-sizing 设置盒子的模型

3.10 弹性盒子

  • 父
    • justify-content
    • align-items 设置子元素的对齐方式
    • flex-wrap 设置是否换行
    • flex-direction 设置 方向
    • align-content 设置各个行的对齐方式
  • 子
    • order 设置 数值给子元素排序 默认是0
    • align-self 自身的对齐方式
    • flex-shrink 设置子元素的收缩比率
    • flex 设置flex的大小,占比权重

3.11 @media 媒介查询,设置响应式的

@media screen and (max-width: 699px) and (min-width: 520px) {
    ul li a {
        padding-left: 30px;
        background: url(email-icon.png) left center no-repeat;
    }
}

四、标准盒模型 和 ie盒模型

  • 标准盒模型
    • box-sizing content-box
    • 宽高就是content内容的宽高
  • ie盒模型
    • box-sizing border-box
    • 宽高包含了padding+border
